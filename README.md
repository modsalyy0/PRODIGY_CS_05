# PRODIGY CS Task-05 - Network Packet Analyzer
This project is a simple Network Packet Analyzer built using Python.

It captures incoming network packets and displays useful packet information for educational purposes.

## Features

- Capture live network packets
- Display packet data
- Analyze raw traffic
- Beginner-friendly cybersecurity project

## Technologies Used

- Python
- Socket Programming

## How to Run

1. Open terminal
2. Run:

python packet_analyzer.py

3. View captured packets in terminal

## Note

This project is developed for educational and authorized testing purposes only.
Administrator permission may be required on Windows.
